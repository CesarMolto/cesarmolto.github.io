PROJECT STARFALL

Project Starfall is an arcade 2D game where the main goal is to survive the longest possible avoiding being hit by the evil aliens or the fast asteroids falling from the sky.

Platform: Browser
Project length: 2 months


PLAY IN BROWSER

OVERVIEW

This project started as a challenge to make a simple and funny video game from scratch in my spare time, and explore interesting concepts such as game feel or game juice.

The original idea of Project Starfall was inspired by the LUDUM DARE games of Brakeys (LUDUM DARE 38) and Blackthornprod (LUDUM DARE 40).

I developed Project Starfall because I wanted users to feel like they are playing an old arcade game but somehow different thanks to my personal touch. For that reason, Project Starfall mixes an old pixel art style with a glow shader filter that makes the game feel really different and impressive.

WHAT I DID

• Created all the art assets.

• Created all the sound effects (but the music).

• Coded all the game mechanics.

• Created a glow shader to add an original style to the game.

• Designed the User Interface and concept of the game.
